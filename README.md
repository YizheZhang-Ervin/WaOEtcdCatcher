# etcd-catcher

## Features
```
- Call Etcd API from Browser
- Invoke WebSSH from Browser
```

## Technologies
```
- Vue with axios,xterm
- Express with axios,ssh2,ws
```

## CMD
```
# Initialize
npm init
npm i express body-parser axios morgan ssh2 ws utf8
npm run dev

# Test
npm install
npm run dev

# Production
npm run start

# Release
git push --tags
```
